# Temperature Converter

A simple temperature converter web application that converts:

- Celsius to Fahrenheit
- Fahrenheit to Celsius

This project was created while learning HTML, CSS, and JavaScript.

## Features
- User-friendly interface
- Real-time temperature conversion
- Supports Celsius and Fahrenheit conversion

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter a temperature value
2. Select the conversion type
3. View the converted temperature
